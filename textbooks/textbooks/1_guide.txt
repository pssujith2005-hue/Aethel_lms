### 2. Conceptual Core Breakdown

#### 2.1 What are Integers?

Integers are a collection of numbers that include all the whole numbers (0, 1, 2, 3, ...) and their negatives (-1, -2, -3, ...). They can be formally defined as the set of numbers $\{\dots, -3, -2, -1, 0, 1, 2, 3, \dots\}$.

*   **Positive Integers**: These are the natural numbers (1, 2, 3, ...). They are greater than zero.
*   **Negative Integers**: These are numbers less than zero (-1, -2, -3, ...). They are the opposites of the positive integers.
*   **Zero**: Zero is an integer that is neither positive nor negative. It acts as the reference point between positive and negative numbers.

**Analogy**: Think of a thermometer. Temperatures above 0°C are positive integers, temperatures below 0°C are negative integers, and 0°C itself is zero. Another analogy is elevation: above sea level is positive, below sea level is negative, and sea level is zero.

#### 2.2 Representation on a Number Line

The number line is an invaluable tool for visualizing integers and understanding their order and magnitude.

*   A horizontal line is drawn with a point marked as zero (0) in the middle.
*   Positive integers are marked to the right of zero, at equal intervals (1, 2, 3, ...).
*   Negative integers are marked to the left of zero, at equal intervals (-1, -2, -3, ...).

```
<-----------------|-----|-----|-----|-----|-----|-----|-----|-----|----------------->
                 -4    -3    -2    -1     0     1     2     3     4
```

**Ordering Integers**:
*   Any number to the right on the number line is greater than any number to its left.
*   For example, $2 > -3$, and $-1 > -4$.
*   As you move to the right, numbers increase. As you move to the left, numbers decrease.

#### 2.3 Absolute Value

The absolute value of an integer is its distance from zero on the number line, regardless of direction. It is always a non-negative value. The symbol for absolute value is two vertical bars: $|...|$.

*   $|5| = 5$ (The distance of 5 from 0 is 5 units).
*   $|-5| = 5$ (The distance of -5 from 0 is 5 units).
*   $|0| = 0$

**Real-world example**: If a submarine is at a depth of 500 meters below sea level (-500m) and a bird is flying at an altitude of 500 meters above sea level (+500m), their absolute distances from sea level are both 500 meters.

#### 2.4 Operations on Integers

##### 2.4.1 Addition of Integers

**Rules for Addition:**

1.  **Adding integers with the same sign**:
    *   Add their absolute values.
    *   Keep the common sign.
    *   *Example*: $5 + 3 = 8$
    *   *Example*: $(-5) + (-3) = -(5+3) = -8$

2.  **Adding integers with different signs**:
    *   Find the difference between their absolute values (subtract the smaller absolute value from the larger one).
    *   The sum takes the sign of the integer with the larger absolute value.
    *   *Example*: $5 + (-3)$
        *   Absolute values are 5 and 3. Difference is $5-3=2$.
        *   The larger absolute value is 5 (from +5), so the sum is positive.
        *   $5 + (-3) = 2$
    *   *Example*: $(-5) + 3$
        *   Absolute values are 5 and 3. Difference is $5-3=2$.
        *   The larger absolute value is 5 (from -5), so the sum is negative.
        *   $(-5) + 3 = -2$

**Analogy**: Think of "gains" as positive and "losses" as negative.
*   You gain ₹5 and then gain ₹3. Total gain: ₹8 ($5+3=8$).
*   You lose ₹5 and then lose ₹3. Total loss: ₹8 ($(-5)+(-3)=-8$).
*   You gain ₹5 but lose ₹3. Net gain: ₹2 ($5+(-3)=2$).
*   You lose ₹5 but gain ₹3. Net loss: ₹2 ($(-5)+3=-2$).

##### 2.4.2 Subtraction of Integers

Subtracting an integer is the same as adding its additive inverse (its opposite).

*   **Rule**: $a - b = a + (-b)$
*   *Example*: $5 - 3 = 5 + (-3) = 2$
*   *Example*: $3 - 5 = 3 + (-5) = -2$
*   *Example*: $5 - (-3) = 5 + (3) = 8$
*   *Example*: $(-5) - (-3) = (-5) + 3 = -2$

**Key Idea**: Change the subtraction sign to an addition sign, and change the sign of the number being subtracted to its opposite. Then follow the rules for addition.

##### 2.4.3 Multiplication of Integers

**Rules for Multiplication:**

1.  **Multiplying integers with the same sign**:
    *   The product is always positive.
    *   *Example*: $5 \times 3 = 15$
    *   *Example*: $(-5) \times (-3) = 15$

2.  **Multiplying integers with different signs**:
    *   The product is always negative.
    *   *Example*: $5 \times (-3) = -15$
    *   *Example*: $(-5) \times 3 = -15$

**Analogy**: Think of "positive" as "doing" and "negative" as "undoing" or "opposite."
*   "Doing good things" (+ve $\times$ +ve) is good (+ve).
*   "Undoing bad things" (-ve $\times$ -ve) is good (+ve).
*   "Doing bad things" (+ve $\times$ -ve) is bad (-ve).
*   "Undoing good things" (-ve $\times$ +ve) is bad (-ve).

##### 2.4.4 Division of Integers

**Rules for Division:**

Similar to multiplication, the rules for signs in division are:

1.  **Dividing integers with the same sign**:
    *   The quotient is always positive.
    *   *Example*: $15 \div 3 = 5$
    *   *Example*: $(-15) \div (-3) = 5$

2.  **Dividing integers with different signs**:
    *   The quotient is always negative.
    *   *Example*: $15 \div (-3) = -5$
    *   *Example*: $(-15) \div 3 = -5$

**Important Note**: Division by zero is undefined. For any integer $a$, $a \div 0$ is not defined. Also, division of integers does not always result in an integer (e.g., $5 \div 2 = 2.5$, which is not an integer).

#### 2.5 Properties of Integers

Integers exhibit several important properties under addition and multiplication. Understanding these properties helps in simplifying calculations and building a strong mathematical foundation.

1.  **Closure Property**:
    *   **Addition**: If $a$ and $b$ are any two integers, then $a+b$ is also an integer.
        *   Example: $5 + (-3) = 2$ (2 is an integer).
    *   **Subtraction**: If $a$ and $b$ are any two integers, then $a-b$ is also an integer.
        *   Example: $5 - (-3) = 8$ (8 is an integer).
    *   **Multiplication**: If $a$ and $b$ are any two integers, then $a \times b$ is also an integer.
        *   Example: $(-5) \times 3 = -15$ (-15 is an integer).
    *   **Division**: Integers are NOT closed under division.
        *   Example: $5 \div 2 = 2.5$ (not an integer).

2.  **Commutative Property**:
    *   **Addition**: The order of addition does not affect the sum. For any integers $a$ and $b$:
        $$a + b = b + a$$
        *   Example: $5 + (-3) = 2$ and $(-3) + 5 = 2$.
    *   **Multiplication**: The order of multiplication does not affect the product. For any integers $a$ and $b$:
        $$a \times b = b \times a$$
        *   Example: $5 \times (-3) = -15$ and $(-3) \times 5 = -15$.
    *   **Subtraction & Division**: These operations are NOT commutative.
        *   Example (Subtraction): $5 - 3 = 2$, but $3 - 5 = -2$.
        *   Example (Division): $6 \div 3 = 2$, but $3 \div 6 = 0.5$.

3.  **Associative Property**:
    *   **Addition**: The way integers are grouped for addition does not affect the sum. For any integers $a, b,$ and $c$:
        $$a + (b + c) = (a + b) + c$$
        *   Example: $2 + ((-3) + 4) = 2 + 1 = 3$. Also, $(2 + (-3)) + 4 = (-1) + 4 = 3$.
    *   **Multiplication**: The way integers are grouped for multiplication does not affect the product. For any integers $a, b,$ and $c$:
        $$a \times (b \times c) = (a \times b) \times c$$
        *   Example: $2 \times ((-3) \times 4) = 2 \times (-12) = -24$. Also, $(2 \times (-3)) \times 4 = (-6) \times 4 = -24$.
    *   **Subtraction & Division**: These operations are NOT associative.

4.  **Distributive Property of Multiplication over Addition/Subtraction**:
    *   Multiplication distributes over addition and subtraction. For any integers $a, b,$ and $c$:
        $$a \times (b + c) = (a \times b) + (a \times c)$$
        $$a \times (b - c) = (a \times b) - (a \times c)$$
        *   Example (over addition): $2 \times (3 + (-5)) = 2 \times (-2) = -4$.
            *   Using distributive property: $(2 \times 3) + (2 \times (-5)) = 6 + (-10) = -4$.
        *   Example (over subtraction): $5 \times (6 - 2) = 5 \times 4 = 20$.
            *   Using distributive property: $(5 \times 6) - (5 \times 2) = 30 - 10 = 20$.

5.  **Identity Property**:
    *   **Additive Identity**: Zero (0) is the additive identity for integers. Adding 0 to any integer does not change its value.
        $$a + 0 = a = 0 + a$$
        *   Example: $7 + 0 = 7$, and $(-12) + 0 = -12$.
    *   **Multiplicative Identity**: One (1) is the multiplicative identity for integers. Multiplying any integer by 1 does not change its value.
        $$a \times 1 = a = 1 \times a$$
        *   Example: $7 \times 1 = 7$, and $(-12) \times 1 = -12$.

6.  **Inverse Property**:
    *   **Additive Inverse**: For every integer $a$, there exists an additive inverse, denoted as $-a$, such that when added together, they yield the additive identity (0).
        $$a + (-a) = 0$$
        *   Example: The additive inverse of 5 is -5, because $5 + (-5) = 0$.
        *   The additive inverse of -8 is 8, because $(-8) + 8 = 0$.
    *   **Multiplicative Inverse**: Not every integer has a multiplicative inverse that is also an integer (only 1 and -1 do). For a number $a$, its multiplicative inverse is $1/a$. Since $1/a$ is generally not an integer for integer $a \ne \pm 1$, we say integers do not possess a general multiplicative inverse property within the set of integers.

**Question 1:**
Evaluate the following expression step-by-step, showing all calculations and justifying the signs of the results:
$(-15) \div 3 + 4 \times (-5) - (-20)$

**Expert Explanation:**

To evaluate this expression, we must follow the order of operations (PEMDAS/BODMAS): Parentheses/Brackets, Exponents/Orders, Multiplication and Division (from left to right), Addition and Subtraction (from left to right).

1.  **Multiplication and Division (from left to right):**
    *   First, perform the division: $(-15) \div 3$.
        *   Here, we have a negative integer divided by a positive integer. According to the rules of division, the quotient will be negative.
        *   $15 \div 3 = 5$. So, $(-15) \div 3 = -5$.
    *   Next, perform the multiplication: $4 \times (-5)$.
        *   Here, we have a positive integer multiplied by a negative integer. According to the rules of multiplication, the product will be negative.
        *   $4 \times 5 = 20$. So, $4 \times (-5) = -20$.

    Substituting these results back into the expression, we get:
    $-5 + (-20) - (-20)$

2.  **Addition and Subtraction (from left to right):**
    *   First, simplify the subtraction of a negative number: $- (-20)$.
        *   Subtracting a negative number is equivalent to adding its positive counterpart (additive inverse). So, $- (-20)$ becomes $+ 20$.

    The expression now is:
    $-5 + (-20) + 20$

    *   Next, perform the addition: $-5 + (-20)$.
        *   We are adding two integers with the same negative sign. We add their absolute values and keep the common negative sign.
        *   $5 + 20 = 25$. So, $-5 + (-20) = -25$.

    The expression now is:
    $-25 + 20$

    *   Finally, perform the addition: $-25 + 20$.
        *   We are adding two integers with different signs. We find the difference between their absolute values ($25 - 20 = 5$). The sign of the result is the sign of the integer with the larger absolute value (which is -25, so negative).
        *   $-25 + 20 = -5$.

**Final Answer:** $-5$

**Question 2:**
A submarine descends 250 meters from sea level, then ascends 120 meters, and finally descends another 100 meters. If sea level is represented by 0 meters, what is the final position of the submarine? Express your answer as an integer and explain the meaning of its sign.

**Expert Explanation:**

Let's represent the vertical movement of the submarine using integers.
*   Movement downwards (descent) will be represented by negative integers.
*   Movement upwards (ascent) will be represented by positive integers.
*   Sea level is our reference point, 0 meters.

1.  **Initial position**: The submarine starts at sea level, which is 0 meters.

2.  **First movement (descends 250 meters)**:
    *   This is a downward movement, so we represent it as $-250$ meters.
    *   Current position: $0 + (-250) = -250$ meters.

3.  **Second movement (ascends 120 meters)**:
    *   This is an upward movement, so we represent it as $+120$ meters.
    *   Current position: $-250 + 120$.
        *   Adding integers with different signs: Subtract the smaller absolute value from the larger ($250 - 120 = 130$). The sign is that of the larger absolute value (from -250, so negative).
        *   Current position: $-130$ meters.

4.  **Third movement (descends another 100 meters)**:
    *   This is a downward movement, so we represent it as $-100$ meters.
    *   Final position: $-130 + (-100)$.
        *   Adding integers with the same (negative) sign: Add their absolute values ($130 + 100 = 230$) and keep the common negative sign.
        *   Final position: $-230$ meters.

**Final Answer:** The final position of the submarine is **-230 meters**.

**Meaning of the sign:**
The negative sign indicates that the submarine's final position is **below sea level**. Specifically, it is 230 meters below sea level.

**Question 3:**
Consider the integers $a = -7$, $b = 3$, and $c = -2$.
Verify if the distributive property of multiplication over addition, $a \times (b + c) = (a \times b) + (a \times c)$, holds true for these integers. Show your work for both sides of the equation.

**Expert Explanation:**

To verify the distributive property, we need to calculate the value of both the left-hand side (LHS) and the right-hand side (RHS) of the equation $a \times (b + c) = (a \times b) + (a \times c)$ using the given integer values and show that they are equal.

Given integers: $a = -7$, $b = 3$, $c = -2$.

**Left-Hand Side (LHS): $a \times (b + c)$**

1.  First, calculate the expression inside the parentheses: $(b + c)$
    *   $b + c = 3 + (-2)$
    *   Adding integers with different signs: Subtract the smaller absolute value from the larger ($3 - 2 = 1$). The sign is that of the larger absolute value (from +3, so positive).
    *   $3 + (-2) = 1$

2.  Now, multiply $a$ by the result: $a \times (1)$
    *   $a \times 1 = -7 \times 1$
    *   Multiplying any integer by the multiplicative identity (1) results in the integer itself.
    *   $-7 \times 1 = -7$

Therefore, LHS = $-7$.

**Right-Hand Side (RHS): $(a \times b) + (a \times c)$**

1.  First, calculate the product $(a \times b)$:
    *   $a \times b = (-7) \times 3$
    *   Multiplying integers with different signs: The product is negative.
    *   $7 \times 3 = 21$. So, $(-7) \times 3 = -21$.

2.  Next, calculate the product $(a \times c)$:
    *   $a \times c = (-7) \times (-2)$
    *   Multiplying integers with the same (negative) sign: The product is positive.
    *   $7 \times 2 = 14$. So, $(-7) \times (-2) = 14$.

3.  Finally, add the two products: $(-21) + 14$
    *   Adding integers with different signs: Subtract the smaller absolute value from the larger ($21 - 14 = 7$). The sign is that of the larger absolute value (from -21, so negative).
    *   $(-21) + 14 = -7$.

Therefore, RHS = $-7$.

**Conclusion:**
Since LHS = $-7$ and RHS = $-7$, we have verified that $a \times (b + c) = (a \times b) + (a \times c)$ for the given integers. The distributive property holds true.
